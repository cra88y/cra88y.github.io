Unicord

Unicord is a one page web application built to clone the appearance and basic functionality of Discord. Built on a React/Flask stack, Unicord uses SQLAlchemy for relational database management and SocketIO for live chat rooms that include the ability to update and delete messages in real time.
[ GITHUB | LIVE SITE ]

InfiniNote

InfiniNote is a minimalist take on Evernote. It features note taking with autosave, the option to organize notes into notebooks, and full managment of existing notes. InfiniNote was built using a React/Express stack. I used React with Redux in order to manage a global state that powers responsive and intuitive fully styled React components.
[ GITHUB | LIVE SITE ]

Planstagram

Plantstagram is website intended to clone the core functionality of Instagram. It is a social networking platform that allows users to create secure accounts to then post pictures, view a feed of posts from users they're following, as well as comment & like picutres across the site.
[ GITHUB | LIVE SITE ]

Knick Knack Overflow

Knick Knack Overflow is created to emulate the basic functionality of stack overflow. Users can sign up then post, answer, and vote on questions. It uses an Express backend and Pug as the templating engine to create a responsive user experience using only vanilla JavaScript.
