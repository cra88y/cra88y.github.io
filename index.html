Unicord

Unicord is a single page web application built to clone the appearance and basic functionality of Discord. Built on a React/Flask stack, Unicord uses SQLAlchemy for relational database management and web sockets for live chat rooms with the ability to update and delete messages in real time. This also serves to decrease request overhead and latency by circumventing traditional HTTP.

InfiniNote
